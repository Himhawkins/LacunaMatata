Selected Projects

VLA-Powered Agentic Framework

  • Developed a mobile manipulator (lift, differential drive, 5-DOF arm) controlled by a Vision-Language Model (VLA).
  • Engineered an agentic framework to create, schedule, and execute new tasks based on user text input (Segmented VLA).
  • Solved top-down navigation by integrating a Gemini vision model with A* path planning for automated VLM control.
  • Utilized local VLMs (Moondream) to solve search and interaction problems with described objects or people.

Generalizable Imitation Learning (SO-100 Arm)

  • Built an HPC pipeline (LeRobot) to train and deploy advanced imitation models (ACT, Diffusion, Groot) on an SO-100 arm.
  • Achieved successful generalization on complex tasks like 'making a sandwich' and 'kitchen cleanup' from teleoperated data.

LLM-VLM Agent for Long-Horizon Robotic Task Decomposition

  • Engineered a multi-modal agent using LLM-based function calling and VLM scene understanding (from an extrinsic camera) to schedule and execute long-horizon tasks like multi-object pick-and-place and navigation.

Hybrid BC-RL Model for Corrective Imitation Learning

  • Enhanced a VQBET Behavioral Cloning model with a residual Reinforcement Learning (RL) layer to significantly boost imitation accuracy and enable corrective behaviors.

Mobile Pick-and-Place via Imitation Learning

  • Applied imitation learning to an SO-100 mobile manipulator to perform complex, multi-stage service tasks (e.g., "get a coke from the kitchen," "clean the table") from a single command.

Decentralized UAV Swarm (DRDO Challenge)

  • Developed an ad-hoc networked, decentralized swarm of 4 drones for an autonomous vision-based search and rescue task using MAVROS and the OLSR mesh routing protocol.

Heterogeneous Swarm Robotics (DRDO Challenge, 1st Runner Up)

  • Developed a heterogeneous swarm of robots with multi-terrain coverage, capable of semi-autonomous SLAM-based building surveys.

Autonomous Indoor Drone (Flipkart GRiD 2.0, Stage 3 Finalist)

  • Implemented a VIO framework using fused stereo camera and IMU data for GPS-denied indoor localization and devised a waypoint generator to autonomously traverse windows.

Decentralized Swarm Algorithms (Kilobots & Modular Bots)

  • Deployed decentralized algorithms (ID allotment, shape formation) on 10 K-team Kilobots and designed prototypes for self-assembling modular bots.

Low-Cost 3D LiDAR Design

  • Engineered a novel 3D LiDAR by fusing a spinning 1D LiDAR, a Kalman Filter-stabilized IMU, and an absolute rotary encoder to achieve low-cost 3D point cloud mapping.

High-Accuracy WiFi RSSI Indoor Localization

  • Designed a WiFi-enabled, ESP-based localization system to triangulate a ground bot's position within a 10 cm error margin using orientation-specific RSSI-to-distance mapping.

ANN for Structural Damage Detection

  • Developed an ANN to solve a 2D elliptic PDE (Inverse Cauchy problem) from partial boundary data, successfully detecting structural damage and material inhomogeneity.

MPPI-Based Optimal Control

  • Designed and simulated a Model Predictive Path Integral (MPPI) controller for complex quadcopter maneuvers (target states, flips) and 3-DOF robotic arm end-effector control.

Vision-Based Gerontechnology

  • Built a `cv2-dnn` based face-tracking Raspberry Pi module to detect falls/accidents and trigger fail-safes.

Auto-Panorama Image Stitcher
